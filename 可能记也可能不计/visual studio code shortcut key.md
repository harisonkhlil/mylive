# visual studio code shortcut key

#vscode #shortcut_keys

| shortcut key         | effect               |
| -------------------- | -------------------- |
| control + option + n | Run code             |
| control + option + ` | Open terminal        |
| control + Shift + f  | Format code          |
| Command + Shift + e  | Open file_tree       |
| Command + 1          | Return the  mainpage |
